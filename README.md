# NewsCloud
之前写的一个新闻app，界面高仿简书，修改了一下，分享出来，有点乱，可供参考。

新闻API来源于APIStore,由于没有各大新闻媒体的正式授权，这种APP是不能上架的，(实测)。

一共四五十个新闻频道，可以个性化用户新闻频道，可以获取用户地址,提供天气信息,还有其他迷之API...<br>

数据库用了coreData，为了区分,数据模型还是用的自己创建的<br>

UI是自己PS调色的，也挺美观的<br>

不合理的地方也不少，可以优化，仅供参考<br>
